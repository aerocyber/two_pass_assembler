TPA: Two Pass Algorithm

TPA is a two pass assembler implementation for the SIC Architecture. It has a few design specification for language but otherwise compatible with the SIC instruction set.

TPA offers a GUI implementation for the assembler. It works on Windows and Linux adn is supported on Android (version <= 11).
